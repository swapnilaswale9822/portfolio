The Platform

AgentLab

No-code/low-code AI agent platform for building, training, and deploying self-improving agents and multi-agent teams. Powers all my products.

ServicePortWhat it does
Agent Service8005Agent orchestration (Agno), teams, workflows, Composio/MCP tools, KB, sessions, memory, Learning Loop
Knowledge Base Service80005 KB types (D1, D3, Semantic, Graph, Hybrid). 50+ formats, vector/keyword/agentic RAG, GraphRAG
Job Service8001Celery async processing: parse, chunk, embed, index. OCR, entity extraction, web scraping
Connector Service8004PostgreSQL, MySQL, MongoDB, DuckDB, BigQuery. Composio tools, MCP servers, triggers/webhooks

What makes AgentLab unique

  • Agent Learning Loop — Experience capture → reflection → wisdom patterns → instruction injection. No competitor has this.
  • Triple KB — D1 (agent core), D3 (user shared), Semantic models (NL-to-SQL).
  • Knowledge Graph + GraphRAG — Neo4j + PgVector, entity/relationship extraction.
  • 5 Responsible AI guardrails — PII, prompt injection, toxicity, topic control, keyword blocking.
  • Visual workflow canvas — Multi-agent orchestration, conditional logic, parallel execution.
  • 950+ tools — Composio SDK + MCP + custom.

Stack at a glance

Frontends: React, Vite, Tailwind, Polaris, Preact. Backends: FastAPI, Python 3.12, Celery, Redis. AI: Agno, OpenAI, Anthropic, Gemini, Groq, Cohere. Data: PostgreSQL (pgvector), Neo4j, Redis. Infra: Docker, Nginx, Sentry, Stripe.

Philosophy

What I'm Building With AI

One thesis: AI agents should learn, remember, and get better — like human employees.

1
Self-Improving Agents
Learning Loop captures experience, reflects, extracts wisdom patterns, injects back into instructions.
2
Multi-Agent Teams
Specialist teams, not monolithic chatbots — NyayaSetu's 5-agent legal team, ShopBrain sales team.
3
Knowledge-First AI
Triple KB + Knowledge Graphs so agents reason over structured, domain-specific knowledge.
4
AI for India
NyayaSetu (11 languages), Kissan Voice Agent, Orgakawach — AI for the underserved.
5
AI Visibility as a category
Geovantax and Shopify AEO — "how visible is your brand to AI?" as important as SEO.
6
Vertical SaaS on one platform
AgentLab is horizontal; Geovantax, AIVentry, ShopBrain, NyayaSetu are verticals. Build once, launch fast.
